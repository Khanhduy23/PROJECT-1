HTML HOME

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu chuẩn được sử dụng để tạo cấu trúc của các trang web. HTML tạo nên nền tảng của mọi trang web, giúp trình duyệt hiểu cách hiển thị nội dung và định dạng văn bản, , Liên kết, và các thành phần khác vv...♪♪

Example HTML Code 1

Với trình soạn thảo "Tự thử" của chúng tôi, bạn có thể chỉnh sửa mã HTML và xem kết quả:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
</body>
</html>

Example HTML Code 2

Đây là một bố cục cơ bản của một trang web bao gồm header, nav, aside, article, footer.

<!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Layout Example</title>
        <style>
            header {
                background-color: #4CAF50;
                color: white;
                padding: 20px;
                text-align: center;
            }
            nav {
                background-color: #333;
                overflow: hidden;
            }
            section, article, aside, footer {
                padding: 20px;
                margin: 10px 0;
            }
            section {
                background-color: #f4f4f4;
            }
            article {
                background-color: #ddd;
            }
            aside {
                background-color: #ccc;
            }
            footer {
                background-color: #444;
                color: white;
            }
        </style>
    </head>
    <body>
        <header>
            <h1>Header Section</h1>
        </header>
    
        <nav>
            <ul>
                <li>Home</li>
                <li>About</li>
                <li>Services</li>
                <li>Contact</li>
            </ul>
        </nav>
    
        <section>
            <article>
                <h2>Article Section</h2>
                <p>This is an article inside the section.</p>
            </article>
            <aside>
                <h2>Aside Section</h2>
                <p>This is an aside inside the section.</p>
            </aside>
        </section>
    
        <footer>
            <p>Footer Section</p>
        </footer>
    </body>
    </html>

Còn rất nhiều thứ bạn sẽ cần học sắp tới , hãy kiên trì và cố gắng nhiều hơn.
