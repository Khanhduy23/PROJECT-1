HTML HOME

Welcome to the HTML HOME section. Here you will find information about the basic structure of HTML, including the use of elements, attributes, and various tags.

Example HTML Code 1

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Example 1</title>
</head>
<body>
    <h1>Hello World from Example 1!</h1>
    <p>This is a paragraph for example 1.</p>
</body>
</html>
