HTML HOME

HTML (HyperText Markup Language) chính là nền tảng của mọi trang web. Tưởng tượng bạn đang xây dựng một ngôi nhà, HTML chính là bộ khung giúp bạn đặt ra vị trí của từng phòng, từng cửa sổ. Nó giúp trình duyệt hiểu cách hiển thị văn bản, , liên kết, và các thành phần khác trên trang một cách chuẩn xác. Bạn sẽ dễ dàng tạo nên cấu trúc web bằng cách sử dụng HTML để mọi thứ hiển thị như ý muốn.♪♪

Example HTML Code 1

Khám phá sự sáng tạo của bạn với trình soạn thảo "Tự thử" độc đáo của chúng tôi! Tại đây, bạn có thể thoải mái chỉnh sửa mã HTML và xem ngay kết quả biến đổi trước mắt.

Khi bạn hoàn tất việc lập trình, chỉ cần nhấn save để lưu lại những sáng tạo của mình trong không gian code cá nhân. Bạn có thể tham khảo và tìm cảm hứng từ các đoạn mã mẫu bên dưới. Hãy nhấn Expand để mở rộng khung chứa mã và theo dõi phần xương của trang web. Nếu bạn muốn thu gọn lại, chỉ cần nhấn Collapse!

Đừng quên nhiệm vụ của bạn: nhấn vào Code và viết mã dựa trên những ví dụ có sẵn. Hãy tự tay trải nghiệm và hiểu rõ hơn về lập trình!

Lưu ý: Sau khi hoàn thành, hãy nhớ lưu lại tác phẩm của bạn. Nếu không, tất cả sẽ trở về trạng thái ban đầu, chỉ còn lại mã mẫu mà thôi!

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
</body>
</html>

Example HTML Code 2

Đây là bố cục cơ bản của một trang web bao gồm các phần: header, nav, aside, article, và footer. Nếu dài quá bạn có thể copy và paste vào khung chứa code

<!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Layout Example</title>
        <style>
            body {
                font-family: Arial, sans-serif;
                margin: 0;
                padding: 0;
            }
            header {
                background-color: #4CAF50;
                color: white;
                padding: 20px;
                text-align: center;
            }
            nav {
                background-color: #333;
                overflow: hidden;
            }
            nav ul {
                list-style-type: none;
                margin: 0;
                padding: 0;
            }
            nav ul li {
                float: left;
                padding: 14px 20px;
                color: white;
                cursor: pointer;
            }
            nav ul li:hover {
                background-color: #575757;
            }
            section, article, aside, footer {
                padding: 20px;
                margin: 10px 0;
            }
            section {
                display: flex;
                justify-content: space-between;
                background-color: #f4f4f4;
            }
            article, aside {
                width: 48%;
            }
            article {
                background-color: #ddd;
            }
            aside {
                background-color: #ccc;
            }
            footer {
                background-color: #4CAF50;
                color: white;
                text-align: center;
            }
        </style>
    </head>
    <body>
        <header>
            <h1>Website Title</h1>
        </header>
        <nav>
            <ul>
                <li><a href="#">Home</a></li>
                <li><a href="#">About</a></li>
                <li><a href="#">Contact</a></li>
            </ul>
        </nav>
        <section>
            <article>
                <h2>Article Title</h2>
                <p>This is the main content area.</p>
            </article>
            <aside>
                <h2>Sidebar Title</h2>
                <p>This is a sidebar area.</p>
            </aside>
        </section>
        <footer>
            <p>Footer content here.</p>
        </footer>
    </body>
</html>

Giới thiệu về không gian code

Trong không gian code, bạn sẽ được trải nghiệm ba phần chính:

  1. Phần HTML: Nơi bạn sẽ thỏa sức sáng tạo và lập trình mã HTML để xây dựng cấu trúc cho trang web của mình.
  2. Phần CSS: Tại đây, bạn có thể tùy chỉnh phong cách và giao diện của trang bằng cách sử dụng mã CSS, biến những ý tưởng của bạn thành hiện thực. (sẽ học ở chương sau)
  3. Phần JavaScript: Đây là không gian để bạn thêm các tính năng tương tác, giúp trang web trở nên sống động và hấp dẫn hơn. (sẽ học ở chương sau)
