HTML HOME

HTML (HyperText Markup Language) chính là nền tảng của mọi trang web. Tưởng tượng bạn đang xây dựng một ngôi nhà, HTML chính là bộ khung giúp bạn đặt ra vị trí của từng phòng, từng cửa sổ. Nó giúp trình duyệt hiểu cách hiển thị văn bản, , liên kết, và các thành phần khác trên trang một cách chuẩn xác. Bạn sẽ dễ dàng tạo nên cấu trúc web bằng cách sử dụng HTML để mọi thứ hiển thị như ý muốn.♪♪

Example HTML Code 1

Với trình soạn thảo "Tự thử" của chúng tôi, bạn có thể chỉnh sửa mã HTML và xem kết quả ngay lập tức:

Khi bạn tự code xong nhấn save kết quả sẽ được lưu lại trong không gian code đó của bạn.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
</body>
</html>

Example HTML Code

Đây là bố cục cơ bản của một trang web bao gồm các phần: header, nav, aside, article, và footer.

<!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Layout Example</title>
        <style>
            body {
                font-family: Arial, sans-serif;
                margin: 0;
                padding: 0;
            }
            header {
                background-color: #4CAF50;
                color: white;
                padding: 20px;
                text-align: center;
            }
            nav {
                background-color: #333;
                overflow: hidden;
            }
            nav ul {
                list-style-type: none;
                margin: 0;
                padding: 0;
            }
            nav ul li {
                float: left;
                padding: 14px 20px;
                color: white;
                cursor: pointer;
            }
            nav ul li:hover {
                background-color: #575757;
            }
            section, article, aside, footer {
                padding: 20px;
                margin: 10px 0;
            }
            section {
                display: flex;
                justify-content: space-between;
                background-color: #f4f4f4;
            }
            article, aside {
                width: 48%;
            }
            article {
                background-color: #ddd;
            }
            aside {
                background-color: #ccc;
            }
            footer {
                background-color: #444;
                color: white;
                text-align: center;
            }
            /* Responsive Design */
            @media (max-width: 600px) {
                section {
                    flex-direction: column;
                }
                article, aside {
                    width: 100%;
                }
                nav ul li {
                    float: none;
                    text-align: center;
                }
            }
        </style>
    </head>
    <body>
        <header>
            <h1>Header Section</h1>
        </header>
    
        <nav>
            <ul>
                <li>Home</li>
                <li>About</li>
                <li>Services</li>
                <li>Contact</li>
            </ul>
        </nav>
    
        <section>
            <article>
                <h2>Article Section</h2>
                <p>This is an article inside the section.</p>
            </article>
    
            <aside>
                <h2>Aside Section</h2>
                <p>This is an aside inside the section.</p>
            </aside>
        </section>
    
        <footer>
            <p>Footer Section</p>
        </footer>
    </body>
</html>
