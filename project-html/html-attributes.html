HTML Attributes

Thuộc tính HTML được sử dụng để cung cấp thêm thông tin về phần tử HTML. Chúng có thể ảnh hưởng đến cách mà phần tử đó hiển thị và tương tác với người dùng.

<a href="https://example.com" target="_blank">Liên kết mới</a>
<img src="image.jpg" alt="Mô tả hình ảnh" width="300">
<input type="text" placeholder="Nhập tên của bạn">

Phân Tích Các Thuộc Tính

<a href="https://example.com" target="_blank">: Thuộc tính href xác định URL mà liên kết sẽ dẫn đến. Thuộc tính target với giá trị "_blank" mở liên kết trong tab mới.

<img src="image.jpg" alt="Mô tả hình ảnh" width="300">: Thuộc tính src chỉ định nguồn hình ảnh, alt cung cấp mô tả thay thế cho hình ảnh, và width điều chỉnh chiều rộng hiển thị của hình ảnh.

<input type="text" placeholder="Nhập tên của bạn">: Thuộc tính type xác định loại input (ở đây là văn bản), và placeholder cung cấp hướng dẫn cho người dùng về thông tin cần nhập.

Một số thuộc tính phổ biến khác bao gồm:

  • class: Chỉ định một hoặc nhiều lớp CSS cho phần tử.
  • id: Xác định một định danh duy nhất cho phần tử, thường dùng cho JavaScript và CSS.
  • style: Áp dụng các kiểu CSS trực tiếp cho phần tử, nhưng không được khuyến khích sử dụng vì làm giảm khả năng bảo trì.
  • disabled: Ngăn người dùng tương tác với phần tử, thường dùng cho các trường nhập liệu.
  • value: Xác định giá trị của các phần tử nhập liệu như input và textarea.
